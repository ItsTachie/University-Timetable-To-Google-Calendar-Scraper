# University Timetable to Google Calendar

A Python-based tool to scrape your University of Liverpool timetable, handle Duo Two-Factor Authentication, and convert the schedule into a `.ics` format compatible with Google and Apple Calendars.

---

## Prerequisites

- **Python 3.x**: Ensure Python is installed.
- **pip**: Python’s package installer (included by default with Python).

---

## Setup & Installation

### 1. Clone the Repository

Open your terminal or bash and run:

```bash
git clone https://github.com/TomCampbellOulton/University-Timetable-To-Google-Calendar-Scraper.git
cd University-Timetable-To-Google-Calendar-Scraper
```

### 2. Install Dependencies

This project uses Playwright for browser automation. Install the library and the required browser binaries:

```bash
pip install playwright
playwright install
```

### 3. Configure Your Settings

Open `config.json` in a text editor and update the following:

- **Credentials:** Replace `YOUR_USERNAME` and `YOUR_PASSWORD` with your university login.
- **Filenames:** You can change `"filename"` (CSV) or `"ics_output"` (ICS) if you want a custom name.
- **Format:** Set `subject_format` to:
  - `"code-title"` (e.g., COMP202 - Complexity of Algorithms)
  - `"code"` (e.g., COMP202)
  - `"title"` (e.g., Complexity of Algorithms)

---

## Usage Instructions

### Step 1: Scrape the Data

Run the main scraper script:

```bash
python __main__.py
```

> **Note:** When the browser opens, you will need to approve the login on your **Duo Authentication app**. Click "Yes, this is my device" when prompted.

### Step 2: Convert to Calendar Format

Once the scraper finishes and creates your CSV, run the converter:

```bash
python csv_to_ics.py
```

---

## How to Import to Your Calendar

### For Google Calendar

1. Open [Google Calendar](https://calendar.google.com/).
2. Click the **+** next to "Other calendars" and select **Create new calendar**. Name it "University Timetable" (this makes it easy to toggle on/off or delete later).
3. Go to **Settings (cog icon)** > **Import & Export**.
4. Click **Select file from your computer** and choose your generated `.ics` file.
5. In the **Add to calendar** dropdown, select the new "University Timetable" calendar you just created.
6. Click **Import**.

### For Apple Calendar

1. Open the Calendar app.
2. Go to **File > Import**.
3. Select the `.ics` file generated by the script.
4. Choose the calendar you wish to add the events to.
